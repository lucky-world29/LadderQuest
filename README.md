# LadderQuest

LadderQuest is a full-stack word ladder puzzle platform where players transform one word into another by changing a single letter at a time. The application leverages graph theory and Breadth-First Search (BFS) to discover the shortest transformation paths while providing an engaging experience through authentication, daily challenges, streak tracking, leaderboards, achievements, and player statistics.

## Features

- User Registration & Authentication (JWT)
- Interactive Word Ladder Challenges
- Shortest Path Discovery using BFS
- Daily Challenges
- Streak Tracking
- Global Leaderboards
- Achievement System
- Player Profiles & Statistics
- Real-time Updates

## Tech Stack

### Frontend

- HTTP Client

### Algorithms & Concepts
- Graph Theory
- Breadth-First Search (BFS)
- Shortest Path Algorithms
- State Management
- RESTful APIs

## Project Goal

The goal of LadderQuest is to combine modern full-stack development practices with algorithmic problem solving by creating a platform that is both educational and engaging for players.
